This program is one that, as an application, enables the use of alternate DNS
providers. Further, it optionally sets up a proxy using the great polipo or
tinyproxy applications. For more information on the latter, please see
http://www.pps.univ-paris-diderot.fr/~jch/software/polipo/ and
https://banu.com/tinyproxy/).
